# 🚀 LeetMetric

LeetMetric is a simple web application that allows users to view and analyze their **LeetCode problem-solving statistics** by entering their LeetCode username. The application fetches real-time data using the **LeetCode GraphQL API** and presents it in a clean dashboard with progress indicators and submission statistics.

---

## 📌 Features

- 🔍 Search any **LeetCode username**
- 📊 Visual progress indicators for **Easy, Medium, and Hard** problems
- 📈 Displays **total submission statistics**
- ⚡ Fetches real-time data using **LeetCode GraphQL API**
- 🧠 Username validation using **Regex**
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **LeetCode GraphQL API**

---

## ⚙️ How It Works

1. User enters their **LeetCode username**
2. The application validates the username format
3. A **GraphQL request** is sent to the LeetCode API
4. The response data is parsed
5. The UI updates with:
   - Problem difficulty progress
   - Submission statistics
